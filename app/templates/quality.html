Качество и запуск

Проверки проекта

Проект проверяется автоматическими тестами, сборкой Docker-образа, нагрузочным сценарием и метриками для мониторинга.

Неделя 12. Тестирование

  • Unit tests для collector и пользовательских функций.
  • Integration tests для API и авторизованных сценариев.
  • Coverage threshold: 40%.
  • Команда: pytest

Неделя 13. Нагрузочное тестирование

  • k6 сценарий для главной, поиска, тем, обзора и healthcheck.
  • Профиль нагрузки: 100+ одновременных пользователей.
  • Отчет: RPS, latency, error rate.
  • Файл: load_tests/k6_public_api.js

Неделя 14. Мониторинг и деплой

  • Prometheus metrics endpoint: /api/v1/metrics
  • Prometheus config и Grafana-ready Docker Compose.
  • Логи приложения выводятся в stdout контейнера.
  • Demo deployment: Vercel + Supabase PostgreSQL.

Неделя 15. Финальная защита

  • Живая демонстрация поиска и автозагрузки из OpenAlex.
  • Показ Supabase БД и аналитики.
  • Показ GitHub Actions и результатов k6.
  • Показ метрик мониторинга.
